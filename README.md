## 2. Описание проекта
**TaskBoard** — это современное Fullstack-приложение для управления списком задач. Оно представляет собой интерактивную доску, которая позволяет эффективно организовывать рабочий процесс и отслеживать текущие дела.

**Основные возможности:**
* **Просмотр:** Отображение актуального списка задач, хранящихся в базе данных SQLite.
* **Создание:** Добавление новых задач с указанием названия и подробного описания.
* **Управление статусом:** Возможность отмечать задачи как выполненные (с визуальным выделением в интерфейсе).
* **Редактирование:** Быстрое изменение названия или описания задачи без перезагрузки страницы.
* **Удаление:** Очистка доски от выполненных или неактуальных записей.
* **Фильтрация:** Удобный поиск и сортировка задач по категориям «Все», «Активные» и «Выполненные».
